\doxylink{requirements_The}{The} term {\itshape structured} stems from \href{https://tools.ietf.org/html/rfc8259}{\texttt{RFC 8259}}:\+

\begin{quote}
JSON can represent four primitive types (strings, numbers, booleans, and null) and two structured types (objects and arrays). \end{quote}


Note that though strings are containers in C++, they are treated as primitive values in JSON.\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2is__structured_autotoc_md550}{}\doxysection{\texorpdfstring{Examples}{Examples}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2is__structured_autotoc_md550}
