\begin{DoxyItemize}
\item \doxylink{requirements_The}{The} comparison always yields {\ttfamily \#!cpp false} if (1) either operand is discarded, or (2) either operand is {\ttfamily NaN} and the other operand is either {\ttfamily NaN} or any other number.

!!! note "{}\+Comparing \`{}\+Na\+N\`{}"{} \begin{DoxyVerb}`NaN` values are unordered within the domain of numbers.
The following comparisons all yield `#!cpp false`:
  1. Comparing a `NaN` with itself.
  2. Comparing a `NaN` with another `NaN`.
  3. Comparing a `NaN` and any other number.
\end{DoxyVerb}
