This macro evaluates to the full name of the {\ttfamily \doxylink{namespacenlohmann}{nlohmann}} namespace.\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2macros_2nlohmann__json__namespace_autotoc_md1290}{}\doxysection{\texorpdfstring{Default definition}{Default definition}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2macros_2nlohmann__json__namespace_autotoc_md1290}
\doxylink{requirements_The}{The} default value consists of the root namespace ({\ttfamily \doxylink{namespacenlohmann}{nlohmann}}) and an inline ABI namespace. See \href{../../features/namespace.md\#structure}{\texttt{{\ttfamily nlohmann} Namespace}} for details.

When the macro is not defined, the library will define it to its default value. Overriding this value has no effect on the library.\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2macros_2nlohmann__json__namespace_autotoc_md1291}{}\doxysection{\texorpdfstring{Examples}{Examples}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2macros_2nlohmann__json__namespace_autotoc_md1291}
