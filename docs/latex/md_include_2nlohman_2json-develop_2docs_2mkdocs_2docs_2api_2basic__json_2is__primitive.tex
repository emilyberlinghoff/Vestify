\doxylink{requirements_The}{The} term {\itshape primitive} stems from \href{https://tools.ietf.org/html/rfc8259}{\texttt{RFC 8259}}:\+

\begin{quote}
JSON can represent four primitive types (strings, numbers, booleans, and null) and two structured types (objects and arrays). \end{quote}


This library extends primitive types to binary types, because binary types are roughly comparable to strings. Hence, {\ttfamily is\+\_\+primitive()} returns {\ttfamily \#!cpp true} for binary values.\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2is__primitive_autotoc_md535}{}\doxysection{\texorpdfstring{Examples}{Examples}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2is__primitive_autotoc_md535}
