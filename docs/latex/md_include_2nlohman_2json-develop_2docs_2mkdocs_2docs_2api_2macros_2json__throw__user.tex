Controls how exceptions are handled by the library.


\begin{DoxyEnumerate}
\item This macro overrides \href{https://en.cppreference.com/w/cpp/language/try_catch}{\texttt{{\ttfamily \#!cpp catch}}} calls inside the library. \doxylink{requirements_The}{The} argument is the type of the exception to catch. As of version 3.\+8.\+0, the library only catches {\ttfamily std::\+out\+\_\+of\+\_\+range} exceptions internally to rethrow them as \href{../../home/exceptions.md\#out-of-range}{\texttt{{\ttfamily json::\+out\+\_\+of\+\_\+range}}} exceptions. \doxylink{requirements_The}{The} macro is always followed by a scope.
\item This macro overrides {\ttfamily \#!cpp throw} calls inside the library. \doxylink{requirements_The}{The} argument is the exception to be thrown. Note that {\ttfamily JSON\+\_\+\+THROW\+\_\+\+USER} should leave the current scope (e.\+g., by throwing or aborting), as continuing after it may yield undefined behavior.
\item This macro overrides {\ttfamily \#!cpp try} calls inside the library. It has no arguments and is always followed by a scope.
\end{DoxyEnumerate}\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2macros_2json__throw__user_autotoc_md1246}{}\doxysection{\texorpdfstring{Parameters}{Parameters}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2macros_2json__throw__user_autotoc_md1246}
