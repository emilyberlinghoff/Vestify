Creates a \href{http://jsonpatch.com}{\texttt{JSON Patch}} so that value {\ttfamily source} can be changed into the value {\ttfamily target} by calling \doxysectlink{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2patch}{{\ttfamily patch}}{0} function.

For two JSON values {\ttfamily source} and {\ttfamily target}, the following code yields always {\ttfamily \#!cpp true}:\+ 
\begin{DoxyCode}{0}
\DoxyCodeLine{source.patch(diff(source,\ target))\ ==\ target;}

\end{DoxyCode}
