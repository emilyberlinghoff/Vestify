Exceptions can be switched off by defining the symbol {\ttfamily JSON\+\_\+\+NOEXCEPTION}. When defining {\ttfamily JSON\+\_\+\+NOEXCEPTION}, {\ttfamily \#!cpp try} is replaced by {\ttfamily \#!cpp if (true)}, {\ttfamily \#!cpp catch} is replaced by {\ttfamily \#!cpp if (false)}, and {\ttfamily \#!cpp throw} is replaced by {\ttfamily \#!cpp std::\+abort()}.

\doxylink{requirements_The}{The} same effect is achieved by setting the compiler flag {\ttfamily -\/fno-\/exceptions}.\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2macros_2json__noexception_autotoc_md1231}{}\doxysection{\texorpdfstring{Default definition}{Default definition}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2macros_2json__noexception_autotoc_md1231}
By default, the macro is not defined.
