Creates a JSON array value from a given initializer list. That is, given a list of values {\ttfamily a, b, c}, creates the JSON value {\ttfamily \#!json \+[a, b, c]\+}. If the initializer list is empty, the empty array {\ttfamily \#!json \+[]\+} is created.\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2array_autotoc_md86}{}\doxysection{\texorpdfstring{Parameters}{Parameters}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2array_autotoc_md86}

This function is only needed to express two edge cases that cannot be realized with the initializer list constructor (\doxysectlink{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2basic__json}{{\ttfamily basic\+\_\+json(initializer\+\_\+list\+\_\+t, bool, value\+\_\+t)}}{0}). These cases are:\+


\begin{DoxyEnumerate}
\item creating an array whose elements are all pairs whose first element is a string -- in this case, the initializer list constructor would create an object, taking the first elements as keys
\item creating an empty array -- passing the empty initializer list to the initializer list constructor yields an empty object
