Detailed Description

Comprehensive financial data structure for individual stocks.

The Stock struct contains all financial information needed for stock analysis, organized into logical categories including identity, market data, income statement, cash flow, balance sheet, ratios, and valuation metrics. All monetary values are in USD, and ratios are expressed as decimals (e.g., 0.15 for 15%).
