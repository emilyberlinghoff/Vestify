Serializes a given JSON value {\ttfamily j} to a byte vector using the BJData (Binary JData) serialization format. BJData aims to be more compact than JSON itself, yet more efficient to parse.


\begin{DoxyEnumerate}
\item Returns a byte vector containing the BJData serialization.
\item Writes the BJData serialization to an output adapter.
\end{DoxyEnumerate}
