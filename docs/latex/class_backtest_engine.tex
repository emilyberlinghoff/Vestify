\doxysubsection{Detailed Description}
Simple backtest engine with periodic rebalancing. 

The engine simulates a portfolio that periodically re-\/ranks the stock universe with the active scoring strategy, selects the highest-\/ranked holdings, and tracks the resulting equity curve across time.
