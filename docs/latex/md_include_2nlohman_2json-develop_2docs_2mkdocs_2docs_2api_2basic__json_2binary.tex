\begin{DoxyEnumerate}
\item Creates a JSON binary array value from a given binary container.
\item Creates a JSON binary array value from a given binary container with subtype.
\end{DoxyEnumerate}

Binary values are part of various binary formats, such as CBOR, Message\+Pack, and BSON. This constructor is used to create a value for serialization to those formats.\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2binary_autotoc_md137}{}\doxysection{\texorpdfstring{Parameters}{Parameters}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2binary_autotoc_md137}
{\ttfamily init} (in) :\+ container containing bytes to use as a binary type

{\ttfamily subtype} (in) :\+ subtype to use in CBOR, Message\+Pack, and BSON\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2binary_autotoc_md138}{}\doxysection{\texorpdfstring{Return value}{Return value}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2binary_autotoc_md138}

Linear in the size of {\ttfamily init}; constant for {\ttfamily typename binary\+\_\+t::\+container\+\_\+type\&\& init} versions.\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2binary_autotoc_md141}{}\doxysection{\texorpdfstring{Notes}{Notes}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2binary_autotoc_md141}
Note, this function exists because of the difficulty in correctly specifying the correct template overload in the standard value ctor, as both JSON arrays and JSON binary arrays are backed with some form of a {\ttfamily std::\+vector}. Because JSON binary arrays are a non-\/standard extension, it was decided that it would be best to prevent automatic initialization of a binary array type, for backwards compatibility and so it does not happen on accident.\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2binary_autotoc_md142}{}\doxysection{\texorpdfstring{Examples}{Examples}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2binary_autotoc_md142}
