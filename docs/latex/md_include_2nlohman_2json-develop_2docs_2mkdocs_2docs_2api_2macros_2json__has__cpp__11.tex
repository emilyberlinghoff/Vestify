\doxylink{requirements_The}{The} library targets C++11, but also supports some features introduced in later C++ versions (e.\+g., {\ttfamily std::\+string\+\_\+view} support for C++17). For these new features, the library implements some preprocessor checks to determine the C++ standard. By defining any of these symbols, the internal check is overridden and the provided C++ version is unconditionally assumed. This can be helpful for compilers that only implement parts of the standard and would be detected incorrectly.\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2macros_2json__has__cpp__11_autotoc_md1205}{}\doxysection{\texorpdfstring{Default definition}{Default definition}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2macros_2json__has__cpp__11_autotoc_md1205}
