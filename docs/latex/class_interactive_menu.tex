\doxysubsection{Detailed Description}
Main user interface class for the Vestify application. 

The \doxylink{class_interactive_menu}{Interactive\+Menu} class provides the interactive terminal interface for the stock analysis application. It coordinates between user input, data operations, and display output through the \doxylink{class_stock_repository}{Stock\+Repository} and various display components. It acts as the application\textquotesingle{}s orchestration layer, delegating specialized work to screening, watchlist, scoring, printing, and backtesting modules.
