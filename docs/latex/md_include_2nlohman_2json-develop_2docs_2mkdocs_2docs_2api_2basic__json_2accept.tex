Checks whether the input is valid JSON.


\begin{DoxyEnumerate}
\item Reads from a compatible input.
\item Reads from a pair of character iterators

\doxylink{requirements_The}{The} value\+\_\+type of the iterator must be an integral type with a size of 1, 2, or 4 bytes, which will be interpreted respectively as UTF-\/8, UTF-\/16, and UTF-\/32.
\end{DoxyEnumerate}

Unlike the \doxysectlink{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2parse}{{\ttfamily parse()}}{0} function, this function neither throws an exception in case of invalid JSON input (i.\+e., a parse error) nor creates diagnostic information.\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2accept_autotoc_md75}{}\doxysection{\texorpdfstring{Template parameters}{Template parameters}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2accept_autotoc_md75}
{\ttfamily Input\+Type} :\+ \doxylink{class_a}{A} compatible input, for instance:\+ \begin{DoxyVerb}- an `std::istream` object
- a `#!c FILE` pointer (throws if null)
- a C-style array of characters
- a pointer to a null-terminated string of single byte characters (throws if null)
- a `std::string`
- an object `obj` for which `begin(obj)` and `end(obj)` produces a valid pair of iterators.
\end{DoxyVerb}


{\ttfamily Iterator\+Type} :\+ a compatible iterator type, for instance. \begin{DoxyVerb}- a pair of `std::string::iterator` or `std::vector<std::uint8_t>::iterator`
- a pair of pointers such as `ptr` and `ptr + len`
\end{DoxyVerb}
