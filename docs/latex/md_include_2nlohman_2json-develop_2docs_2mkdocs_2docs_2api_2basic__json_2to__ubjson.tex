Serializes a given JSON value {\ttfamily j} to a byte vector using the UBJSON (Universal Binary JSON) serialization format. UBJSON aims to be more compact than JSON itself, yet more efficient to parse.


\begin{DoxyEnumerate}
\item Returns a byte vector containing the UBJSON serialization.
\item Writes the UBJSON serialization to an output adapter.
\end{DoxyEnumerate}
