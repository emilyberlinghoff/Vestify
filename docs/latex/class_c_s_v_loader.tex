\doxysubsection{Detailed Description}
CSV loader for stock datasets. 

Reads the project\textquotesingle{}s fundamentals CSV format, normalizes headers, parses numeric fields into a {\ttfamily \doxylink{struct_stock}{Stock}} record, and reports recoverable row-\/level errors without aborting the whole import.
