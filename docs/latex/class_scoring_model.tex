\doxysubsection{Detailed Description}
Abstract base for scoring models. 

Each concrete model computes a single numeric score for a stock based on its own criteria (value, growth, momentum, etc.). This abstraction allows strategies to combine interchangeable factor models without coupling ranking logic to any one implementation.
