\doxysubsection{Detailed Description}
Utility class for formatting and displaying stock information. 

The \doxylink{class_stock_printer}{Stock\+Printer} class provides static methods for displaying stock data in various formats including detailed stock information, summary lists, and ranked displays. It includes formatting utilities for monetary values and colorized display of financial metrics based on performance thresholds.

All methods are static, making this a utility class that doesn\textquotesingle{}t require instantiation. Output is directed to stdout with ANSI color codes for enhanced readability in terminal environments.
