This macro indicates whether the compiler and standard library support 3-\/way comparison. Possible values are {\ttfamily 1} when supported or {\ttfamily 0} when unsupported.\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2macros_2json__has__three__way__comparison_autotoc_md1223}{}\doxysection{\texorpdfstring{Default definition}{Default definition}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2macros_2json__has__three__way__comparison_autotoc_md1223}
\doxylink{requirements_The}{The} default value is detected based on the preprocessor macros {\ttfamily \#!cpp \+\_\+\+\_\+cpp\+\_\+impl\+\_\+three\+\_\+way\+\_\+comparison} and {\ttfamily \#!cpp \+\_\+\+\_\+cpp\+\_\+lib\+\_\+three\+\_\+way\+\_\+comparison}.

When the macro is not defined, the library will define it to its default value.\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2macros_2json__has__three__way__comparison_autotoc_md1224}{}\doxysection{\texorpdfstring{Examples}{Examples}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2macros_2json__has__three__way__comparison_autotoc_md1224}
