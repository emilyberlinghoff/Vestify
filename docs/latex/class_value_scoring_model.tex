\doxysubsection{Detailed Description}
Scores stocks on value characteristics. 

Evaluates four value dimensions:\+
\begin{DoxyItemize}
\item P/\+E ratio (lower is better for value investors)
\item Dividend yield (higher signals shareholder returns)
\item EV/\+\+FCF ratio (lower means cheaper relative to cash generation)
\item FCF yield (higher means more free cash flow per dollar of market cap)
\end{DoxyItemize}

Each component maps to a 0-\/100 sub-\/score, then averaged. The model is intended to favor cheaper cash-\/generating businesses while still rewarding direct shareholder return through dividends.
