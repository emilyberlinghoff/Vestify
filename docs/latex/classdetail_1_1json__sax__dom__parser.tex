\subsubsection*{template$<$typename Basic\+Json\+Type, typename Input\+Adapter\+Type$>$\newline
class detail::\+json\+\_\+sax\+\_\+dom\+\_\+parser$<$ Basic\+Json\+Type, Input\+Adapter\+Type $>$}
SAX implementation to create a JSON value from SAX events. 

This class implements the \doxylink{structjson__sax}{json\+\_\+sax} interface and processes the SAX events to create a JSON value which makes it basically a DOM parser. \doxylink{requirements_The}{The} structure or hierarchy of the JSON value is managed by the stack {\ttfamily \doxylink{classdetail_1_1json__sax__dom__parser_ac10a461ec4eb10f9242eda0ebf8145b8}{ref\+\_\+stack}} which contains a pointer to the respective array or object for each recursion depth.

After successful parsing, the value that is passed by reference to the constructor contains the parsed value.
