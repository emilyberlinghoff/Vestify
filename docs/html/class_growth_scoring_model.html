Detailed Description

Scores stocks on growth characteristics.

Evaluates four growth dimensions:

  • P/E ratio (higher signals market expects future earnings growth)
  • Operating margin (higher margin supports scalable growth)
  • ROE (high return on equity indicates efficient growth)
  • Low dividend yield (earnings reinvested into the business)

Each component maps to a 0-100 sub-score, then averaged. It is designed to reward companies whose fundamentals suggest reinvestment capacity and scalable operating performance.

Calculate a growth score for one stock.

Maps growth-oriented valuation and profitability measures onto the shared 0-100 scoring scale, then averages the components that are available and sensible for the current stock.

Parameters
stockStock fundamentals to evaluate.
Returns
Growth score in the range [0, 100].

Higher P/E signals market expects growth. Map P/E 0-50 -> 0-100.

Operating margin: higher is better for scalable businesses. Stored as fraction. Map 0-0.40 (40%) -> 0-100.

ROE: higher signals efficient use of equity to generate growth. Stored as fraction. Map 0-0.40 (40%) -> 0-100.

Low dividend yield implies earnings reinvested for growth.
