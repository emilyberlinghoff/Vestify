\doxysubsection{Detailed Description}
Handles loading and saving watchlists to disk. 

Uses JSON files for persistence. The repository hides file-\/format details from the rest of the application so the UI and domain logic can work directly with {\ttfamily \doxylink{class_watch_list}{Watch\+List}} objects.
