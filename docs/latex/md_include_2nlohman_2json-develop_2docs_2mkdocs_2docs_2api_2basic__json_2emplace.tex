Inserts a new element into a JSON object constructed in-\/place with the given {\ttfamily args} if there is no element with the key in the container. If the function is called on a JSON null value, an empty object is created before appending the value created from {\ttfamily args}.\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2emplace_autotoc_md233}{}\doxysection{\texorpdfstring{Template parameters}{Template parameters}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2emplace_autotoc_md233}
