\doxylink{requirements_The}{The} merge patch format is primarily intended for use with the HTTP PATCH method as a means of describing a set of modifications to a target resource\textquotesingle{}s content. This function applies a merge patch to the current JSON value.

\doxylink{requirements_The}{The} function implements the following algorithm from Section 2 of \href{https://tools.ietf.org/html/rfc7396}{\texttt{RFC 7396 (JSON Merge Patch)}}:\+
