Vestify is a C++17 stock analysis application for Linux/\+\+WSL that loads bundled stock fundamentals from CSV, lets users screen and rank stocks interactively, manages persistent watchlists, and runs historical backtests with periodic rebalancing.

The current application launches through a single interactive terminal menu. The older command-\/line argument mode has been removed.

\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md__r_e_a_d_m_e_autotoc_md2}

\begin{DoxyItemize}
\item Load bundled stock fundamentals from {\ttfamily data/\+nasdaq100\+\_\+fundamentals\+\_\+full.\+csv}
\item Screen stocks with multiple criteria at once
\item Choose sectors from a generated list instead of typing them manually
\item Print the current screening filters before running a screen
\item Rank stocks with five factor models:\+ Value, Growth, Momentum, Quality, and Efficiency
\item Use predefined strategy presets or create custom scoring weights at runtime
\item Inspect per-\/factor score breakdowns for ranked stocks
\item Manage multiple persistent watchlists stored as JSON
\item Run historical backtests with periodic rebalancing
\item Refresh the fundamentals CSV with the bundled Python update script
\end{DoxyItemize}

\hypertarget{md__r_e_a_d_m_e_autotoc_md15}{}\doxysection{\texorpdfstring{Optional Environment Variables}{Optional Environment Variables}}\label{md__r_e_a_d_m_e_autotoc_md15}
Vestify can fetch historical price data from Alpha Vantage for backtesting. To enable that, create a {\ttfamily .env} file in the project root:\+


\begin{DoxyCode}{0}
\DoxyCodeLine{ALPHAVANTAGE\_API\_KEY=your\_api\_key\_here}

\end{DoxyCode}


The {\ttfamily .env} file is already ignored by Git and should not be committed.

To load it manually in a Linux/\+\+WSL shell:\+


\begin{DoxyCode}{0}
\DoxyCodeLine{set\ -\/a}
\DoxyCodeLine{source\ .env}
\DoxyCodeLine{set\ +a}

\end{DoxyCode}


If you do not set an API key, the app can still:\+


\begin{DoxyItemize}
\item load the bundled CSV
\item screen stocks
\item rank stocks
\item manage watchlists
\end{DoxyItemize}

Backtesting that depends on missing historical data may fail until cached price files exist or an API key is provided.

\hypertarget{md__r_e_a_d_m_e_autotoc_md27}{}\doxysection{\texorpdfstring{Scoring Models}{Scoring Models}}\label{md__r_e_a_d_m_e_autotoc_md27}
Vestify includes five factor models:\+


\begin{DoxyTable}{3}{}{}{1}
\SetCell{c,bg=\tableheadbgcolor,font=\bfseries}Model  &\SetCell{c,bg=\tableheadbgcolor,font=\bfseries}Main Focus  &\SetCell{c,bg=\tableheadbgcolor,font=\bfseries}Example Metrics  \\
Value  &Cheap valuation and cash generation  &{\ttfamily P/\+E}, dividend yield, {\ttfamily EV/\+\+FCF}, {\ttfamily FCF yield}  \\
Growth  &Reinvestment and scalable profitability  &{\ttfamily P/\+E}, operating margin, {\ttfamily ROE}, dividend yield  \\
Momentum  &Profitability and cash-\/flow strength proxies  &{\ttfamily ROA}, {\ttfamily FCF yield}, net margin  \\
Quality  &Balance-\/sheet health and operating quality  &current ratio, debt-\/to-\/equity, gross margin, {\ttfamily ROE}  \\
Efficiency  &Capital and operating efficiency  &{\ttfamily ROA}, operating margin, {\ttfamily EV/\+\+EBIT}  \\
\end{DoxyTable}


Predefined strategy presets combine those models with different weights, and users can also create a custom weighting interactively.

\begin{DoxyItemize}

\item max {\ttfamily P/\+S}

\item min {\ttfamily FCF yield}
\item min {\ttfamily ROE}
\item min operating margin
\item max debt-\/to-\/equity
\item min current ratio
\item min dividend yield
\end{DoxyItemize}

\begin{DoxyItemize}

\item printing the current active filter set before running the screen

\hypertarget{md__r_e_a_d_m_e_autotoc_md35}{}\doxysection{\texorpdfstring{Notes}{Notes}}\label{md__r_e_a_d_m_e_autotoc_md35}

\begin{DoxyItemize}
\item The project is intended for Linux/\+\+WSL use.
\item {\ttfamily build/\+}, {\ttfamily .env}, and virtual environments are ignored by Git.
\item Bundled fundamentals data allows most of the application to work offline.
\item Historical price files are stored under {\ttfamily data/\+historical/\+}.
