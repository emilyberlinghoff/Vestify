Header for the \doxylink{class_stock_repository}{Stock\+Repository} class that manages stock data collections. 

This header defines the \doxylink{class_stock_repository}{Stock\+Repository} class which serves as the central data store for stock information in the Vestify application. It provides methods for loading stock data from CSV files and retrieving stocks by various criteria.
