\doxysubsection{Detailed Description}
Entry point for the Vestify stock analysis application. 

This file contains the main function that initializes and runs the interactive menu system for the stock analysis application. The application provides functionality for loading stock data, performing analysis, and displaying results through an interactive terminal menu.
