\end{DoxyEnumerate}\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2insert_autotoc_md449}{}\doxysection{\texorpdfstring{Iterator invalidation}{Iterator invalidation}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2insert_autotoc_md449}
For all cases where an element is added to an {\bfseries{array}}, a reallocation can happen, in which case all iterators (including the \doxysectlink{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2end}{{\ttfamily end()}}{0} iterator) and all references to the elements are invalidated. Otherwise, only the \doxysectlink{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2end}{{\ttfamily end()}}{0} iterator is invalidated. Also, any iterator or reference after the insertion point will point to the same index, which is now a different value.

For \doxysectlink{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2ordered__json}{{\ttfamily ordered\+\_\+json}}{0}, also adding an element to an {\bfseries{object}} can yield a reallocation which again invalidates all iterators and all references. Also, any iterator or reference after the insertion point will point to the same index, which is now a different value.\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2insert_autotoc_md450}{}\doxysection{\texorpdfstring{Parameters}{Parameters}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2insert_autotoc_md450}
{\ttfamily pos} (in) :\+ iterator before which the content will be inserted; may be the {\ttfamily end()} iterator

{\ttfamily val} (in) :\+ value to insert

{\ttfamily first} (in) :\+ the start of the range of elements to insert
