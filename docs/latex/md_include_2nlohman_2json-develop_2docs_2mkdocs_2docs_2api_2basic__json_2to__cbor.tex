Serializes a given JSON value {\ttfamily j} to a byte vector using the CBOR (Concise Binary Object Representation) serialization format. CBOR is a binary serialization format that aims to be more compact than JSON itself, yet more efficient to parse.


\begin{DoxyEnumerate}
\item Returns a byte vector containing the CBOR serialization.
\item Writes the CBOR serialization to an output adapter.
\end{DoxyEnumerate}
