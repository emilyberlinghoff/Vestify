\doxylink{requirements_The}{The} folder contains the required files to create a \href{https://kapeli.com/docsets}{\texttt{docset}} which can be used in documentation browsers like \href{https://kapeli.com/dash}{\texttt{Dash}}, \href{https://velocity.silverlakesoftware.com}{\texttt{Velocity}}, or \href{https://zealdocs.org}{\texttt{Zeal}}.

\doxylink{requirements_The}{The} docset can be created with


\begin{DoxyCode}{0}
\DoxyCodeLine{make\ nlohmann\_json.docset}

\end{DoxyCode}


\doxylink{requirements_The}{The} generated folder {\ttfamily nlohmann\+\_\+json.\+docset} can then be opened in the documentation browser.
