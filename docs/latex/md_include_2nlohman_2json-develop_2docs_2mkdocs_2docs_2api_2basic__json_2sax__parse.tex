Read from input and generate SAX events


\begin{DoxyEnumerate}
\item Read from a compatible input.
\item Read from a pair of character iterators

\doxylink{requirements_The}{The} value\+\_\+type of the iterator must be an integral type with a size of 1, 2, or 4 bytes, which will be interpreted respectively as UTF-\/8, UTF-\/16, and UTF-\/32.
\end{DoxyEnumerate}

\doxylink{requirements_The}{The} SAX event lister must follow the interface of \doxysectlink{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2json__sax_2index}{{\ttfamily json\+\_\+sax}}{0}.\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2sax__parse_autotoc_md828}{}\doxysection{\texorpdfstring{Template parameters}{Template parameters}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2sax__parse_autotoc_md828}
{\ttfamily Input\+Type} :\+ \doxylink{class_a}{A} compatible input, for instance:\+ \begin{DoxyVerb}- an `std::istream` object
- a `FILE` pointer
- a C-style array of characters
- a pointer to a null-terminated string of single byte characters
- an object `obj` for which `begin(obj)` and `end(obj)` produces a valid pair of
  iterators.
\end{DoxyVerb}
