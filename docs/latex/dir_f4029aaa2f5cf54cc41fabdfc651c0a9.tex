File \href{nlohmann-json.py}{\texttt{nlohmann-\/json.\+py}} contains a pretty printer for GDB for JSON values of this library. It was originally published as \href{https://gist.github.com/ssbssa/60da5339c6e6036b2afce17de06050ea\#file-nlohmann-json-py}{\texttt{Gist}} by \href{https://github.com/ssbssa}{\texttt{Hannes Domani}}.\hypertarget{dir_f4029aaa2f5cf54cc41fabdfc651c0a9_autotoc_md1951}{}\doxysubsection{\texorpdfstring{How to use}{How to use}}\label{dir_f4029aaa2f5cf54cc41fabdfc651c0a9_autotoc_md1951}

\begin{DoxyItemize}
\item Add line


\begin{DoxyCode}{0}
\DoxyCodeLine{source\ /path/to/nlohmann-\/json.py}

\end{DoxyCode}


to {\ttfamily \texorpdfstring{$\sim$}{\string~}/\+.gdbinit}. Note you must replace {\ttfamily /\+path/\+to} with whatever path you stored file {\ttfamily nlohmann-\/json.\+py}.
\item In GDB, debug as usual. When you want to pretty-\/print a JSON value {\ttfamily var}, type


\begin{DoxyCode}{0}
\DoxyCodeLine{p\ -\/pretty\ on\ -\/array\ on\ -\/-\/\ var}

\end{DoxyCode}
