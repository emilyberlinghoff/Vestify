This function returns a JSON object with information about the library, including the version number and information on the platform and compiler.\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2meta_autotoc_md587}{}\doxysection{\texorpdfstring{Return value}{Return value}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2meta_autotoc_md587}
JSON object holding version information


\begin{DoxyTable}{2}{}{}{1}
\SetCell{c,bg=\tableheadbgcolor,font=\bfseries}key  &\SetCell{c,bg=\tableheadbgcolor,font=\bfseries}description  \\
{\ttfamily compiler}  &Information on the used compiler. It is an object with the following keys:\+ {\ttfamily c++} (the used C++ standard), {\ttfamily family} (the compiler family; possible values are {\ttfamily clang}, {\ttfamily icc}, {\ttfamily gcc}, {\ttfamily ilecpp}, {\ttfamily msvc}, {\ttfamily pgcpp}, {\ttfamily sunpro}, and {\ttfamily unknown}), and {\ttfamily version} (the compiler version).  \\
{\ttfamily copyright}  &\doxylink{requirements_The}{The} copyright line for the library as string.  \\
{\ttfamily name}  &\doxylink{requirements_The}{The} name of the library as string.  \\
{\ttfamily platform}  &\doxylink{requirements_The}{The} used platform as string. Possible values are {\ttfamily win32}, {\ttfamily linux}, {\ttfamily apple}, {\ttfamily unix}, and {\ttfamily unknown}.  \\
{\ttfamily url}  &\doxylink{requirements_The}{The} URL of the project as string.  \\
{\ttfamily version}  &\doxylink{requirements_The}{The} version of the library. It is an object with the following keys:\+ {\ttfamily major}, {\ttfamily minor}, and {\ttfamily patch} as defined by \href{http://semver.org}{\texttt{Semantic Versioning}}, and {\ttfamily \doxylink{classstring}{string}} (the version string).  \\
\end{DoxyTable}
