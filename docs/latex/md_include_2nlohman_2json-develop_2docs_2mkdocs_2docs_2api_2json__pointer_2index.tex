\doxylink{class_a}{A} JSON pointer defines a string syntax for identifying a specific value within a JSON document. It can be used with functions \doxysectlink{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2at}{{\ttfamily at}}{0} and {\ttfamily operator\+[]\+}. Furthermore, JSON pointers are the base for JSON patches.\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2json__pointer_2index_autotoc_md1023}{}\doxysection{\texorpdfstring{Template parameters}{Template parameters}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2json__pointer_2index_autotoc_md1023}
{\ttfamily Ref\+String\+Type} :\+ the string type used for the reference tokens making up the JSON pointer
