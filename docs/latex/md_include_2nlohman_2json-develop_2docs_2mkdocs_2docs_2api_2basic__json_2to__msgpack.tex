Serializes a given JSON value {\ttfamily j} to a byte vector using the Message\+Pack serialization format. Message\+Pack is a binary serialization format that aims to be more compact than JSON itself, yet more efficient to parse.


\begin{DoxyEnumerate}
\item Returns a byte vector containing the Message\+Pack serialization.
\item Writes the Message\+Pack serialization to an output adapter.
\end{DoxyEnumerate}
