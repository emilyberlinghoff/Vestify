This enumeration is used in the \doxysectlink{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2dump}{{\ttfamily dump}}{0} function to choose how to treat decoding errors while serializing a {\ttfamily \doxylink{classbasic__json}{basic\+\_\+json}} value. Three values are differentiated:\+

strict :\+ throw a {\ttfamily type\+\_\+error} exception in case of invalid UTF-\/8

replace :\+ replace invalid UTF-\/8 sequences with U+\+FFFD ({\ucr} REPLACEMENT CHARACTER)

ignore :\+ ignore invalid UTF-\/8 sequences; all bytes are copied to the output unchanged\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2error__handler__t_autotoc_md284}{}\doxysection{\texorpdfstring{Examples}{Examples}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2error__handler__t_autotoc_md284}
