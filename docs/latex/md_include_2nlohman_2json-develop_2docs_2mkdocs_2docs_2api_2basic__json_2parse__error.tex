\doxylink{requirements_The}{The} library throws this exception when a parse error occurs. Parse errors can occur during the deserialization of JSON text, BSON, CBOR, Message\+Pack, UBJSON, as well as when using JSON Patch.

Member {\ttfamily byte} holds the byte index of the last read character in the input file (see note below).

Exceptions have ids 1xx (see \href{../../home/exceptions.md\#parse-errors}{\texttt{list of parse errors}}).
