\item Deserialize from a pair of character iterators

\doxylink{requirements_The}{The} {\ttfamily value\+\_\+type} of the iterator must be an integral type with size of 1, 2, or 4 bytes, which will be interpreted respectively as UTF-\/8, UTF-\/16, and UTF-\/32.

{\ttfamily Input\+Type} :\+ \doxylink{class_a}{A} compatible input, for instance:\+ \begin{DoxyVerb}- an `std::istream` object

\end{DoxyVerb}
