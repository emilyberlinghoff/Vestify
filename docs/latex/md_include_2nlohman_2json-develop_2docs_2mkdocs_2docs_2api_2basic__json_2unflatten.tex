\doxylink{requirements_The}{The} function restores the arbitrary nesting of a JSON value that has been flattened before using the \doxysectlink{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2flatten}{{\ttfamily flatten()}}{0} function. \doxylink{requirements_The}{The} JSON value must meet certain constraints:\+


\begin{DoxyEnumerate}
\item \doxylink{requirements_The}{The} value must be an object.
\item \doxylink{requirements_The}{The} keys must be JSON pointers (see \href{https://tools.ietf.org/html/rfc6901}{\texttt{RFC 6901}})
\item \doxylink{requirements_The}{The} mapped values must be primitive JSON types.
\end{DoxyEnumerate}\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2unflatten_autotoc_md941}{}\doxysection{\texorpdfstring{Return value}{Return value}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2unflatten_autotoc_md941}
