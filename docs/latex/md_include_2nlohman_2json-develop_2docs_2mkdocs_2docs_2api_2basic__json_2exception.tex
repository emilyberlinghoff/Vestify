This class is an extension of \href{https://en.cppreference.com/w/cpp/error/exception}{\texttt{{\ttfamily std::\+exception}}} objects with a member {\ttfamily id} for exception ids. It is used as the base class for all exceptions thrown by the {\ttfamily \doxylink{classbasic__json}{basic\+\_\+json}} class. This class can hence be used as "{}wildcard"{} to catch exceptions, see example below.
