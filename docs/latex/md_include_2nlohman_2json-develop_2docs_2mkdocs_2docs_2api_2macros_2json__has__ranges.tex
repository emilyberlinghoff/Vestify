This macro indicates whether the standard library has any support for ranges. Implies support for concepts. Possible values are {\ttfamily 1} when supported or {\ttfamily 0} when unsupported.\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2macros_2json__has__ranges_autotoc_md1215}{}\doxysection{\texorpdfstring{Default definition}{Default definition}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2macros_2json__has__ranges_autotoc_md1215}
\doxylink{requirements_The}{The} default value is detected based on the preprocessor macro {\ttfamily \#!cpp \+\_\+\+\_\+cpp\+\_\+lib\+\_\+ranges}.

When the macro is not defined, the library will define it to its default value.\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2macros_2json__has__ranges_autotoc_md1216}{}\doxysection{\texorpdfstring{Examples}{Examples}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2macros_2json__has__ranges_autotoc_md1216}
