\doxysubsection{Detailed Description}
Scores stocks on growth characteristics. 

Evaluates four growth dimensions:\+
\begin{DoxyItemize}
\item P/\+E ratio (higher signals market expects future earnings growth)
\item Operating margin (higher margin supports scalable growth)
\item ROE (high return on equity indicates efficient growth)
\item Low dividend yield (earnings reinvested into the business)
\end{DoxyItemize}

Each component maps to a 0-\/100 sub-\/score, then averaged. It is designed to reward companies whose fundamentals suggest reinvestment capacity and scalable operating performance.
