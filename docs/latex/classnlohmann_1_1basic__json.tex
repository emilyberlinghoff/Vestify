checks the class invariants 

This function asserts the class invariants. It needs to be called at the end of every constructor to make sure that created objects respect the invariant. Furthermore, it has to be called each time the type of a JSON value is changed, because the invariant expresses a relationship between {\itshape m\+\_\+type} and {\itshape m\+\_\+value}.

Furthermore, the parent relation is checked for arrays and objects:\+ If {\itshape check\+\_\+parents} true and the value is an array or object, then the container\textquotesingle{}s elements must have the current value as parent.
