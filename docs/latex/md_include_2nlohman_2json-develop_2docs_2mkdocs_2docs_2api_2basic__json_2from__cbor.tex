Deserializes a given input to a JSON value using the CBOR (Concise Binary Object Representation) serialization format.


\begin{DoxyEnumerate}
\item Reads from a compatible input.
\item Reads from an iterator range.
\end{DoxyEnumerate}

\doxylink{requirements_The}{The} exact mapping and its limitations are described on a \doxysectlink{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2features_2binary__formats_2cbor}{dedicated page}{0}.\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2from__cbor_autotoc_md331}{}\doxysection{\texorpdfstring{Template parameters}{Template parameters}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2from__cbor_autotoc_md331}
{\ttfamily Input\+Type} :\+ \doxylink{class_a}{A} compatible input, for instance:\+ \begin{DoxyVerb}- an `std::istream` object
- a `FILE` pointer
- a C-style array of characters
- a pointer to a null-terminated string of single byte characters
