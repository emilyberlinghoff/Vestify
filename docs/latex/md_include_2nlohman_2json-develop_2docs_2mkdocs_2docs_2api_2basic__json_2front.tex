Returns a reference to the first element in the container. For a JSON container {\ttfamily \#!cpp c}, the expression {\ttfamily \#!cpp c.\+front()} is equivalent to {\ttfamily \#!cpp \texorpdfstring{$\ast$}{*}c.begin()}.\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2front_autotoc_md358}{}\doxysection{\texorpdfstring{Return value}{Return value}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2front_autotoc_md358}
In the case of a structured type (array or object), a reference to the first element is returned. In the case of number, string, boolean, or binary values, a reference to the value is returned.\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2front_autotoc_md359}{}\doxysection{\texorpdfstring{Exception safety}{Exception safety}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2front_autotoc_md359}
Strong guarantee:\+ if an exception is thrown, there are no changes in the JSON value.\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2front_autotoc_md360}{}\doxysection{\texorpdfstring{Exceptions}{Exceptions}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2front_autotoc_md360}
If the JSON value is {\ttfamily \#!json null}, exception \href{../../home/exceptions.md\#jsonexceptioninvalid_iterator214}{\texttt{{\ttfamily invalid\+\_\+iterator.\+214}}} is thrown.\hypertarget{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2front_autotoc_md361}{}\doxysection{\texorpdfstring{Complexity}{Complexity}}\label{md_include_2nlohman_2json-develop_2docs_2mkdocs_2docs_2api_2basic__json_2front_autotoc_md361}
