\begin{DoxyEnumerate}
\item Removes an element from a JSON value specified by iterator {\ttfamily pos}. \doxylink{requirements_The}{The} iterator {\ttfamily pos} must be valid and dereferenceable. Thus, the {\ttfamily end()} iterator (which is valid, but is not dereferenceable) cannot be used as a value for {\ttfamily pos}.

If called on a primitive type other than {\ttfamily \#!json null}, the resulting JSON value will be {\ttfamily \#!json null}.
\item Remove an element range specified by {\ttfamily \+[first; last)} from a JSON value. \doxylink{requirements_The}{The} iterator {\ttfamily first} does not need to be dereferenceable if {\ttfamily first == last}:\+ erasing an empty range is a no-\/op.

If called on a primitive type other than {\ttfamily \#!json null}, the resulting JSON value will be {\ttfamily \#!json null}.
\item Removes an element from a JSON object by key.
\item See 3. This overload is only available if {\ttfamily Key\+Type} is comparable with {\ttfamily \#!cpp typename object\+\_\+t::\+key\+\_\+type} and {\ttfamily \#!cpp typename object\+\_\+comparator\+\_\+t::\+is\+\_\+transparent} denotes a type.
\item Removes an element from a JSON array by index.
