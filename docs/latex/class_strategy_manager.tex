\doxysubsection{Detailed Description}
\doxylink{class_strategy_manager}{Strategy\+Manager} — owns presets, active strategy, and ranking logic. 

Manages predefined scoring strategy presets and applies them to stocks.

Satisfies the acceptance tests for "{}\+Apply Predefined Scoring Strategy Presets"{}:\+
\begin{DoxyEnumerate}
\item Multiple predefined strategies exist.
\item Selecting a strategy updates the scoring weights.
\item Re-\/running scoring reflects the new weights.
\item A default strategy is applied on startup (constructor).
\item Switching strategies does not require restarting the application. 
\end{DoxyEnumerate}
